GIT

  • Enter your git notes here
    • git status: checks what branch we are currently on
    • git checkout -b branch-name: creates a new branch and switches to it
    • git clone will clone a repository
    • git commit will stage changes
    • git commit -m "added CSS to project" is an example of how to use this.
    • git push will push changes to the repo
    • git pull will pull changes from the repo
    • git mkdir will create a directory such as a new folder
    • git touch will create a new file within the current directory
    • pwd will show the current directory location
    • cd alone will bring us back to the base or parent directory
    • cd .. will go "up" one directory. This will bring us from, for example, c:/Jacob/users back to c:/Jacob
    • git add -A will add the changes to the memory to be staged afterwards.
